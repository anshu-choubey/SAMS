Usage Guide


Import the Notification System
<script src="../assets/js/notifications.js"></script>
Available Functions
Function Parameters Description
showSuccessDialog() title, message, callback Show a success dialog
showErrorDialog() title, message, callback Show an error dialog
showWarningDialog() title, message, callback Show a warning dialog
showInfoDialog() title, message, callback Show an info dialog
showConfirmDialog() title, message, onConfirm, onCancel Show a confirmation dialog
showLoadingDialog() message Show a loading dialog (returns object with hide() method)
showToast() type, title, message Show a toast notification
Examples
// Success with callback
showSuccessDialog('Success!', 'Department added.', function() {
    location.reload();
});

// Confirmation
showConfirmDialog('Delete?', 'Sure?', 
    function() { console.log('Confirmed'); },
    function() { console.log('Cancelled'); }
);

// Loading
const loader = showLoadingDialog('Processing...');
setTimeout(() => loader.hide(), 3000);

// Toast
showToast('success', 'Done!', 'Operation completed.');
